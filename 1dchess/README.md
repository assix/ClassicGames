# 1D Chess (Python/Pygame)

A unique 1D chess variant played on a single strip. Features King, Rook, and Knight mechanics. AI uses a Greedy Heuristic algorithm (King > Capture > Advance).

This game features custom procedural graphics and requires no external assets.
