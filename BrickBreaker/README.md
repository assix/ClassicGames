# Python Brick Breaker

A classic arcade-style Brick Breaker game built with Python and Pygame.

## Features
- **Dynamic Physics:** Ball angle changes based on paddle impact.
- **Game Loops:** 5 Lives per game.
- **Layouts:** Press `R` to randomize brick layouts (Standard, Pyramid, Random Gaps).
- **Controls:** Pause/Resume functionality.

## Controls
- **Arrow Left / Right:** Move Paddle
- **Space:** Launch Ball / Pause Game
- **R:** Randomize Layout (before launch) / Restart (after Game Over)
