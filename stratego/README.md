# Stratego (Python/Pygame)

High-fidelity Stratego clone. Features a 'Turtle Defense' setup algorithm (Back-row Flag + Bomb perimeter) and a Greedy Heuristic AI that evaluates moves based on Rank Differential and Scout Probing logic.

This game features custom procedural graphics and requires no external assets.
